# DecodeLabs Internship - Project 1

## Rule-Based AI Chatbot

This project is a simple rule-based chatbot developed using Python.

### Features
- Greets users
- Responds to predefined questions
- Handles unknown inputs
- Uses dictionary-based response lookup
- Uses an infinite loop for continuous conversation
- Exits when user types "bye"

### Concepts Used
- Python Dictionaries
- While Loops
- If Statements
- Input Sanitization
- String Methods
